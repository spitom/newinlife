# NEWINLIFE — Understrap Child Theme

Repozytorium zawiera kod motywu potomnego WordPress (Understrap + Bootstrap 5).

## Wymagania

- Git
- Node.js 20
- npm
- XAMPP / lokalny serwer
- VS Code

## Struktura projektu

Repo dotyczy tylko motywu:

wp-content/themes/newinlife-child

## Pierwsze uruchomienie

Po sklonowaniu repo:

npm install
npm run watch

## Workflow

Start pracy:

git pull
npm run watch

Zapis zmian:

git add .
git commit -m "Opis zmian"
git push
